## svg-icon组件

### 引用方式
- 无须引用，已全局注册

### 组件ref类型推断
- 无

### 使用方式
```bash
<svg-icon icon-class="icon-xx-xx" />
```

### 参数
```bash
iconClass // 具体svg名称
className // class样式名称 目前有绑定的几个class： 激活状态：icon-active(显示主色)； 大小：icon-16(宽高均为16px)； 默认色：icon-default-color(自带默认主题色)
```

### 事件
- 无

### 属性
- 无

### 方法
无
